Worksheet Prompts

  1. Name the person and the last real context.
  2. Write the useful reason for the message.
  3. Offer one simple next step.
  4. Remove pressure language before sending.

Pack Checklist

  1. Real prior context confirmed
  2. No fake urgency
  3. One useful reason included
  4. One next step offered
  5. No bulk send configured
