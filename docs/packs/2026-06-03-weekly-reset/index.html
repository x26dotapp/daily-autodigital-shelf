Worksheet Prompts

  1. Write the three things that must not surprise you this week.
  2. List bills, errands, messages, and cleanup tasks.
  3. Choose one task to delegate, defer, or delete.
  4. Write the week-end shutdown rule.

Pack Checklist

  1. Surprises named
  2. Loose ends listed
  3. One task reduced
  4. Shutdown rule written
  5. No overpacked week
