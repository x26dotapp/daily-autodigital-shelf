Worksheet Prompts

  1. Write the appointment time, place, and contact.
  2. List the top three questions.
  3. Name the documents or photos to bring.
  4. Write the follow-up action before leaving.

Pack Checklist

  1. Time and place written
  2. Questions listed
  3. Documents gathered
  4. Follow-up action captured
  5. No medical/legal advice added
