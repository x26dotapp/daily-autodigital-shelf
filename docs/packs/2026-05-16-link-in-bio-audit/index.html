Worksheet Prompts

  1. Write the current top link and what it asks visitors to do.
  2. Remove or park links that distract from the main action.
  3. Write a clearer button label.
  4. Choose the next public proof link.

Pack Checklist

  2. Distracting links parked

  5. No fake metrics added
