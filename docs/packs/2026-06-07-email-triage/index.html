Worksheet Prompts

  1. Search for invoices, appointments, deadlines, and direct requests.
  2. Pick five messages that need action.
  3. Choose reply, schedule, archive, or defer for each.
  4. Write the one reply you can send first.

Pack Checklist

  1. Important searches run
  2. Five messages selected
  3. Action label chosen
  4. First reply drafted
  5. Bulk send avoided
