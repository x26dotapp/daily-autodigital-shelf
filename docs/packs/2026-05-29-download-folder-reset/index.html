Worksheet Prompts

  1. Sort the first 20 files into keep, archive, delete, or review.
  2. Create one named destination folder.
  3. Write what should never be saved here again.
  4. Schedule the next short reset.

Pack Checklist

  1. First 20 files handled
  2. Destination folder made
  3. Delete pile cleared
  4. Review pile named
  5. Next reset scheduled
